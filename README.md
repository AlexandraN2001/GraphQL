# GraphQL + Swagger API
This project is a simple API based on **GraphQL**, documented with **Swagger**, and run in a **Docker** container. It allows you to perform GraphQL queries and view the API documentation via Swagger.

## Technologies Used
- **Node.js** (version 16)
- **Express** for the web server
- **express-graphql** for GraphQL integration
- **swagger-jsdoc** and **swagger-ui-express** for Swagger API documentation
- **Docker** for containerizing the project

## Features
- **GraphQL**:
  - `hello`: Returns a greeting message.
  - `add`: Performs the sum of two integers.

- **Swagger UI**:
  - Provides a visual interface to explore the API and its routes.

## Prerequisites
- Docker installed on your machine.
- Node.js and npm installed if you prefer to run the project outside Docker (optional).

## Instructions for Use

### 1. Clone the Repository
If you don’t have the repository yet, clone it using Git:

```bash
git clone https://github.com/alenac07/graphql-app.git
cd graphql-app
### 2. Run the Project with Docker
Step 1: Build the Docker image
docker build -t graphql-app .
Step 2: Run the Docker container
docker run -p 4000:4000 graphql-app

### 3. Explore the API
GraphQL Playground: Navigate to http://localhost:4000/graphql to interact with the GraphQL API.
Swagger UI: Navigate to http://localhost:4000/api-docs to view the API documentation generated by Swagger.
