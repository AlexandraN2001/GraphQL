# 🚀 GraphQL + Swagger API

Welcome to a simple **GraphQL** API, documented with **Swagger**, and containerized using **Docker**. This project lets you perform GraphQL queries and explore the API documentation via Swagger. 🧑‍💻✨

## 🛠️ Technologies Used
- **Node.js** (version 16) 🚀
- **Express** for the web server 🌐
- **express-graphql** for integrating GraphQL 🔗
- **swagger-jsdoc** & **swagger-ui-express** for Swagger API documentation 📖
- **Docker** for containerizing the project 🐳

## 🔥 Features
- **GraphQL**:
  - `hello`: Returns a friendly greeting message! 😊
  - `add`: Adds two integers and returns the sum ➕🔢

- **Swagger UI**:
  - Provides a visual interface to explore the API and its routes 👀

## ⚙️ Prerequisites
- Docker installed on your machine 🐳
- Node.js and npm installed if you prefer to run the project outside Docker (optional) 💻

## 🏃‍♂️ Instructions for Use

### 1️⃣ Clone the Repository
If you don’t have the repository yet, clone it using Git:

```bash
git clone https://github.com/alenac07/graphql-app.git
cd graphql-app

2️⃣ Run the Project with Docker
🛠️ Step 1: Build the Docker image
docker build -t graphql-app .
🚀 Step 2: Run the Docker container
docker run -p 4000:4000 graphql-app
3️⃣ Explore the API
GraphQL Playground: 🚀 Navigate to http://localhost:4000/graphql to interact with the GraphQL API.
Swagger UI: 📖 Visit http://localhost:4000/api-docs to view the API documentation generated by Swagger.

📂 Project Structure
